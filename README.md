# TODO-app　API

## 内容
TODOアプリのためのAPI

## 仕様
- todo(タスク)の追加、更新、削除
- todoの検索
- assignee(担当者)の追加、更新、削除
- todoへのassigneeの割り当て
- 特定のassigneeに割り当てられたtodoの一覧検索

## 使用方法

```bash
$ git clone && cd TODO-app 
```
